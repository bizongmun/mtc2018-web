# mtc2018-web

## やること

- [ ] MTCインフォメーション画面
- [ ] セッション一覧画面
- [ ] セッション詳細画面

## 使い方

インストール

```sh
$ npm i
```

開発用サーバーを起動

```sh
$ export GRAPHQL_ENDPOINT=your_development_graphql_endpoint
$ npm run dev
```

プロダクションサーバーを起動　

```sh
$ export GRAPHQL_ENDPOINT=your_production_graphql_endpoint
$ npm run build
$ npm run dev
```
